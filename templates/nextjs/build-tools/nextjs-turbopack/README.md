# Next.js + Turbopack Boilerplate

The fastest Next.js development experience with Turbopack.

[![Next.js](https://img.shields.io/badge/Next.js-15-000000?logo=next.js)](https://nextjs.org/)
[![Turbopack](https://img.shields.io/badge/Turbopack-Rust-FF4154)](https://turbo.build/pack)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.7-3178C6?logo=typescript)](https://www.typescriptlang.org/)

## Features

- **Next.js 15** - App Router, Server Components
- **Turbopack** - Rust-based incremental bundler
- **Instant HMR** - Sub-millisecond updates
- **TypeScript** - Full type safety
- **Path Aliases** - Clean `@/` imports

---

## Quick Start

```bash
# Install dependencies
pnpm install

# Start with Turbopack (blazing fast!)
pnpm dev

# Build for production
pnpm build
```

---

## Why Turbopack?

| Metric | Turbopack | Webpack |
|--------|-----------|---------|
| Cold Start | ~200ms | ~2-5s |
| HMR | <10ms | ~200ms |
