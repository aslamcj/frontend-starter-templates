# React + Parcel Boilerplate

A zero-configuration React boilerplate powered by Parcel 2.

[![React](https://img.shields.io/badge/React-19-61DAFB?logo=react)](https://react.dev/)
[![Parcel](https://img.shields.io/badge/Parcel-2-E8C42F?logo=parcel)](https://parceljs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.7-3178C6?logo=typescript)](https://www.typescriptlang.org/)

## Features

- **React 19** - Latest React features
- **Parcel 2** - Zero configuration bundler
- **TypeScript 5.7** - Type safety
- **Automatic transforms** - No config needed
- **Fast builds** - Multi-core compilation
- **Hot reloading** - Instant updates

---

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

---

## Why Parcel?

Parcel requires **zero configuration**:

- Auto-detects TypeScript, JSX, CSS
- Automatic code splitting
- Built-in transformers for most file types
- Multi-core compilation

## Project Structure

```
react-parcel/
├── public/              # Static assets
├── src/
│   ├── components/      # React components
│   ├── hooks/           # Custom hooks
│   ├── styles/          # CSS files
│   └── main.tsx         # Entry point
├── index.html           # HTML entry (source)
└── package.json         # Dependencies
```
