# React + Emotion Boilerplate

Performant CSS-in-JS with the css prop and styled components.

[![React](https://img.shields.io/badge/React-19-61DAFB?logo=react)](https://react.dev/)
[![Emotion](https://img.shields.io/badge/Emotion-11-C43BAD)](https://emotion.sh/)

## Quick Start

```bash
pnpm install && pnpm dev
```

## Usage

### css Prop
```tsx
<div css={{ color: 'blue', fontSize: '2rem' }}>Hello</div>
```

### styled Components
```tsx
const Button = styled.button`
  background: blue;
  color: white;
`;
```
