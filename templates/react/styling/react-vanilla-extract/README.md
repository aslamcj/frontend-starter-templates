# React + Vanilla Extract Boilerplate

Type-safe CSS-in-TypeScript with zero runtime.

[![React](https://img.shields.io/badge/React-19-61DAFB?logo=react)](https://react.dev/)
[![Vanilla Extract](https://img.shields.io/badge/Vanilla_Extract-1.16-pink)](https://vanilla-extract.style/)

## Quick Start

```bash
pnpm install && pnpm dev
```

## Features

- Full TypeScript support
- Zero runtime CSS
- Type-safe theming
- styleVariants for variants
