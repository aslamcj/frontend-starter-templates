# React + shadcn/ui Boilerplate

Beautifully designed accessible components built on Radix UI and Tailwind.

[![React](https://img.shields.io/badge/React-19-61DAFB?logo=react)](https://react.dev/)
[![shadcn/ui](https://img.shields.io/badge/shadcn/ui-Components-000000)](https://ui.shadcn.com/)
[![Tailwind](https://img.shields.io/badge/Tailwind-4-06B6D4?logo=tailwindcss)](https://tailwindcss.com/)

## Quick Start

```bash
pnpm install && pnpm dev
```

## Features

- Accessible by default (Radix UI)
- You own the code
- Tailwind CSS styling
- Dark mode ready

## Adding Components

```bash
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add input
```
